O2. ----------------- WHICH TOPICS WILL BE COVERED IN THE WORKSHOP?.

The workshop is created in such way, the student wont be bored or feel "back to school in summers".

the workshop is taught by trained college students,so communication will not have a generation gap.





O1.


Assembling the robot


Initially the first step, the assembling of the robot,connecting all the parts properly and fix all the lose parts properly.In this students will learn about mechanics and electronics.


O2.


Installation of software


For some interface between the assembled robot and the computer, we will need to install some programs on the computers.

O3.


Computer controlled robot


A simple program to make the robot do some task by giving commands on computer.In this we will learn 2 modules

O4.


IR Sensors And applications


We will see working of IR sensor and also work with its applications.In this we will learn 4 modules.

O5.


Mobile controlled robot & application


We will use mobile keypad to perform tasks on the robot.In this we will learn 2 modules.

O6.


Accelerometer & application


Simple use of accelerometer and complete explaination of the sensor.In this we will learn 3 modules.
